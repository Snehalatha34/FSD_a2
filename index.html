Portfolio

Optimizing Student Placement Prediction Using Ensemble Learning

In this project, we implemented various traditional machine learning models such as Logistic Regression, K-Nearest Neighbors (KNN), Support Vector Machine (SVM), Decision Tree, and Naive Bayes on the Placement_data_full_class dataset. We then applied several ensemble algorithms, including Voting Classifier, Bagging, AdaBoost, Gradient Boosting, and XGBoost. By combining the basic models—Linear Regression, KNN, and SVM—which individually achieved an accuracy of 89%, we enhanced the performance through ensemble learning and achieved an improved accuracy of 91%.

An Intelligent System to Count Students in Large Classroom Using Deep Learning

In this project, we addressed the issue of accurately counting students in large classrooms manually, which may be prone to errors. The utilization of the YOLO v8 model for student head detection and counting has proven to be highly effective and efficient for our project. This advancement holds significant promise for applications in educational settings, allowing for improved monitoring of classroom attendance, engagement, and overall student presence.
